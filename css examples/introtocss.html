9 Types of selector in Css

Universal Selector *{}
for Ex. this we have to write in style tag in header
*{ margin: 0; padding: 0; color: rgb(11, 117, 112); }




Element type Selector p{}
for Ex. this we have to write in style tag in header
p{ color: teal; } h1{ color: teal; }
Here p tag and h1 is a element




ID Selector #error{}
for Ex. this we have to write in style tag in header
#name{ color: turquoise; }
any tag id="name"




Class Selector .blue{}
for Ex. this we have to write in style tag in header
.classname{ color: turquoise; }
any tag class="classname"




Descendant Selector ul.menu li a{}
for Ex. this we have to write in style tag in header
ul.menu a{ text-decoration: none; color: rgb(148, 100, 38); }
its decorate which anchor tag whose unorderlist(ul) class="menu"




Child Selector ul>li{}
for Ex. this we have to write in style tag in header
.menu>a{ text-decoration: none; }
its applicable for which anchor tag which is imidiate child of menu class




Adjacent Sibling Selector h1+p{}
for Ex. this we have to write in style tag in header
h1+p{ text-decoration: underline; }
its applicable for which p tag which is imidiate next under h1 tag




General Sibling Selector h1~p{}
for Ex. this we have to write in style tag in header
h1~p{ text-decoration: underline; }
its applicable for every p tag which is under the h1 tag




Grouping Selector h1,h2,ul,h3{}
for Ex. this we have to write in style tag in header
h1,h2,h3{ color: teal; }
its used for Grouping tag seprated by comma
all the attributes (which are mention in class{}) are applicable for all the tags which is seprated by comma in sytle tag
